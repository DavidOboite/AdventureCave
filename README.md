# Treasure Cave Adventure

This is a command-line adventure game written in Python.

The program reads a hero party file provided as a command-line argument,
allows the player to choose a **Character**, then explores rooms in a treasure cave
with random monsters, treasure, and events.

The player's hero stats are loaded from the input file and affect gameplay.

## Run the game
> python adventure.py heroparty1.txt or python adventure.py heroparty2.txt
 
### With Hero Party 1 Characters or Hero Party 2 Characters
> Choose your hero: 
> 
> Numbers for hero party Characters....Type 1,2,3, or 4.
